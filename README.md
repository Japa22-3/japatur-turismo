## 🌐 Acesso

Visualize o site em: [https://japa22-3.github.io/japatur-turismo](https://japa22-3.github.io/japatur-turismo)

## ⚙️ Como Publicar (GitHub Pages)

1. Vá em **Settings** do repositório
2. Menu lateral: **Pages**
3. Em *Source*, selecione **Deploy from a branch**
4. Escolha `main` → `/root` → clique **Save**
5. Aguarde ~2 min e acesse: `https://japa22-3.github.io/japatur-turismo`
